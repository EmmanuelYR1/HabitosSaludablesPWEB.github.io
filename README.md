# DailyWay - Página de Descarga

Esta es la página web para promocionar y distribuir la aplicación DailyWay.

## Contenido

- **index.html** - Página principal
- **styles.css** - Estilos y diseño
- **script.js** - Funcionalidad interactiva

## Cómo usar

### 1. Agregar tu APK

Coloca tu archivo APK en este mismo directorio con el nombre `app-release.apk`

Si tu APK tiene otro nombre, edita la línea 178 en `index.html`:

```html
<a href="./TU_NOMBRE_APK.apk" class="btn btn-download" download>
```

### 2. Abrir la página

Simplemente abre `index.html` en tu navegador web favorito.

### 3. Personalización

Puedes personalizar fácilmente:

- **Nombre de la app**: Buscar "DailyWay" y reemplazar
- **Colores**: Editar las variables CSS en `styles.css` (líneas 17-28)
- **Contenido**: Editar el texto en `index.html`
- **Imágenes**: Agregar tus propias capturas de pantalla
